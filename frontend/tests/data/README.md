# 测试数据目录

## 目录结构

```
tests/data/
├── human/          # 人工撰写的文本样本
│   ├── chinese/    # 中文人工文本
│   └── english/    # 英文人工文本
└── ai/             # AI 生成的文本样本
    ├── chinese/    # 中文 AI 文本
    └── english/    # 英文 AI 文本
```

## 样本要求

### Human 文本特征
- 自然语言变化
- 不完美的句子结构
- 个人风格明显
- 可能有拼写/语法错误

### AI 文本特征
- 句子长度均匀
- 词汇多样性较低
- 结构过于规整
- 缺乏个人风格

## 样本数量目标
- 每个类别至少 50 个样本
- 每个样本 200-1000 字
- 涵盖不同主题和领域
